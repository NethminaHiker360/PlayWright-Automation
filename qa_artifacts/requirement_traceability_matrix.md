# Requirement Traceability Matrix

| Requirement ID | Description | Covered Test Cases |
| --- | --- | --- |
| R1 | Users must authenticate with username/password to access the dashboard. | TC-01, TC-02, TC-03 |
| R2 | Employee numbers resolve to employee records before allowing actions. | TC-04 |
| R3 | Employees can clock in and clock out with backend confirmation. | TC-05 |
| R4 | Breaks and lunches prevent conflicting activities until ended. | TC-06 |
| R5 | Jobs or tasks can be selected prior to recording activities. | TC-07 |
| R6 | Productive and downtime activities are logged with correct availability rules. | TC-07 |
| R7 | Employees can view self-service My Hours and My Smoko Times details. | TC-08 |
| R8 | Supervisors can monitor active work, downtime, and tasks in Work Tracker. | TC-09 |
| R9 | Report shortcuts open the related Frappe reports. | TC-10 |
| R10 | Logout clears the session and redirects to login. | TC-11 |
